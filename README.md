# 🎬 CinéCrew Backend

Backend Node.js/Express + MongoDB pour l'application **CinéCrew** :
- Authentification (JWT)
- Gestion des profils
- Création et recherche de missions
- Messagerie temps réel (Socket.io)
- Gestion des abonnements (Stripe-ready)
